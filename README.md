# RACER-Oracle

🏎️ **Oracle Race: Path of the Two Rivers**

A real-time, zero-backend multiplayer racing game built with HTML, CSS, JavaScript, and MQTT over WebSockets.

🎮 **[Play the Game Here!](https://rrrmar.github.io/RACER-Oracle/)**

## Features
- **Real-time Multiplayer:** Race against others currently online globally.
- **Backendless Architecture:** Powered entirely by a public MQTT broker (`test.mosquitto.org`). No server database required!
- **Hall of Fame:** Persistent top 10 leaderboard utilizing MQTT Retained Messages.
- **Oracle Coins Betting:** Wager your browser-local coins. The winner takes the entire prize pool!

## How to Play
1. Enter your Oracle Name and set your wager.
2. Click **Enter the Realm**.
3. Tap the **Spacebar** as fast as you can to rush to the finish line!
